# Batalha Naval
  Simples batalha naval feito em C.

# Configuração e compilação
Clone o repositório:
   ```bash
   git clone https://github.com/jvpaz/batalhaNaval-C.git
   ```

Compile o projeto utilizando o *Makefile*
   ```bash
   make all
   ```

Limpe os arquivos gerados, caso seja necessário:
   ```bash
   make clean
   ```
